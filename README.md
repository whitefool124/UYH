# 符印守卫

一个面向毕设演示的浏览器端体感施法游戏 MVP。

## 当前版本已实现

- 浏览器请求摄像头权限
- MediaPipe Hands 实时跟踪单手关键点
- 将手部左右三分区映射为三种法术：冰霜术 / 护盾术 / 火焰术
- 通过稳定停留约 0.5 秒完成一次施法确认
- Canvas 三路线防守战场、敌人推进、得分、结界受损、失败与重开
- 仓库内附带一份快速落地策划：`GAME_PLAN.md`

## 本地运行

```bash
npm install
npm run dev
```

启动后在浏览器打开终端输出的本地地址，通常是：

```text
http://localhost:5173
```

## 操作方式

1. 允许浏览器访问摄像头。
2. 将一只手伸进镜头。
3. 手停留在左 / 中 / 右区域约半秒后，会确认释放对应法术：
   - 左区：冰霜术
   - 中区：护盾术
   - 右区：火焰术
4. 成功击退敌人会得分，漏防会让结界受损。
5. 结界受损达到上限后游戏结束。
6. 离镜后可准备下一次施法；点击右上角按钮可重新开始。

## 版本说明

- 当前版本已不再是“挡板接球”演示，而是“三路线体感施法防守”MVP。
- 交互重点从“连续控制”转为“离散施法 + 稳定确认”，以降低疲劳并增强施法感。
- 当前法术和敌人系统仍保持轻量级，便于后续继续迭代特效、克制关系和关卡节奏。
